# Privacy Policy

Homepage New Tab does not collect, store, transmit, sell, or share personal data.

The extension runs only on `https://homepage.romaine.life/*`. It listens for a local page event containing a URL selected in the homepage UI, validates that the request came from a homepage tab, and asks Chrome to open that URL in a tab.

The extension does not make network requests of its own and does not use analytics, tracking, advertising, or remote code.
